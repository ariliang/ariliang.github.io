    Git 命令

    Posted at — Oct 5, 2019

    记录常用 Git 命令

    配置用户名和邮箱

    添加用户和邮箱

    1$ git config --global user.name "your_name"
2$ git config --global user.email "example@example.com"

    查看已设置的用户

    1$ git config user.name
2$ git config user.email

    创建新的仓库

    1$ echo "# Introduction" >> README.md
2$ git init
3$ touch .gitignore
4$ git add .
5$ git commit -m "log"
6$ git remote add origin [repository_address]
7$ git push -u origin master

    远程仓库相关

    1$ git clone [repo]                              #克隆到本地
2$ git remote -v                                 #查看远程仓库
3$ git remote add [name] [repo]                  #添加远程仓库
4$ git remote rm [name]                          #删除远程仓库
5$ git remote set-url --push [name] [new_repo]   #修改仓库
6$ git pull [remote_repo] [local_branch]         #拉取远程仓库
7$ git push [remote_repo] [local_branch]         #推送到远程仓库

    添加删除缓存区

    1$ git add [file]                #添加将要合并的文件
2$ git add .                     #添加所有
3$ git rm --cached [file]        #删除缓存的文件
4$ git rm -r --cached .          #清楚所有缓存的文件
